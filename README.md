### Required configs:
in `.env` or `Enviroment Variables` of hoster
+ `DB_URI` | MongoDB connection url 
+ `PORT` | Port which application will run (if empty, app will run in port 3000)

### How to run:

```
url_shortner/~ npm install
url_shortner/~ npm run start
```
